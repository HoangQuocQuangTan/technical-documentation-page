Introduction

HTML (HyperText Markup Language) is the standard language for creating web pages and web applications. It is a markup language that uses a system of tags and attributes to define the structure and content of a web page.

HTML contains a set of standard elements for building pages, such as headings, paragraphs, images, links, and tables. Core HTML can be extended for many different purposes by adding other technologies; for example:

  • CSS (Cascading Style Sheets) extends the language to control the presentation of web pages, allowing for complex styling and layout.
  • JavaScript extends the language by providing objects to control the browser and its Document Object Model (DOM), allowing web pages to interact and respond to user actions.

HTML and CSS

HTML and CSS are closely related but play different roles in building websites. HTML (HyperText Markup Language) provides the structure and content of a page, while CSS (Cascading Style Sheets) focuses on presentation and formatting. Although the two are often used together, they have distinct characteristics and purposes.

Unlike HTML, which uses tags to define elements like headings, paragraphs, or links, CSS works through rules to apply styling like color, font, or layout. HTML is static, focusing on “what to say,” while CSS is more dynamic, deciding “how to say it.”

One important difference is that CSS supports inheritance and cascading, allowing styles to be applied in order of precedence or inherited from a parent element. This reduces code duplication and makes it easier to manage a large-scale look and feel. HTML, on the other hand, does not have this mechanism and relies entirely on tag structure. CSS also provides responsive design capabilities through media queries, allowing the web page to adapt to a variety of screen sizes. HTML, while it can incorporate basic attributes like width or height, cannot dynamically adjust to complex layouts without CSS.

Hello world

To get started with writing HTML, create a new file and write your first "Hello World" HTML code:

<!DOCTYPE html>
<html>
<head>
    <title>My First HTML Page</title>
</head>
<body>
    <h1>Hello World!</h1>
    <p>Welcome to your first HTML document.</p>
</body>
</html>

Save the file with a .html extension (e.g., index.html), then open it in your browser to see your webpage come to life!

Basic HTML Tags

In addition to headings and paragraphs, HTML provides many other tags for building web page content. Here are some of the most basic and important tags.

  1. List

    You can create lists in HTML. There are two main types:

    • <ul>: Unordered List, displayed with bullet points.
    • <ol>: Ordered List, displayed by numbers or letters.
    • <li>: List Item, used inside both <ul> and <ol>.

  2. Links

    The <a> tag (short for "anchor") is used to create hyperlinks to other pages or sections on the same page. The most important attribute is the href (hypertext reference), which contains the URL of the link.

  3. Images

    The <img> tag is used to embed an image in a web page. It is a self-closing tag, with no <img> tag. The two required attributes are:

    • src: (Source) Path to the image file.
    • alt (Alternative text) Text that describes the image. This text will be displayed if the image fails to load and is important for visually impaired users (using screen readers) and for SEO.

Class and Id

id and class are two of the most important global attributes in HTML. They do not change how an element is displayed but are used for identification and grouping, primarily for styling (CSS) and interaction (JavaScript).

  1. id

    The id attribute is used to specify a unique identifier for an HTML element.

    • Rule: The value of an id must be used only once per page.
    • Usage: It is typically used to target a single, specific element with CSS or JavaScript.

  2. class

    The class attribute is used to group one or more elements together.

    • Rule: The same class value can be used on multiple elements. An element can also have multiple classes, separated by spaces.
    • Usage: It is commonly used to apply the same style to a group of elements.
